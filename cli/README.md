# Agor CLI Tools

This directory contains the `agor-bg` and `agor-prompt` CLI tools.

## Tools

### `agor-bg`
Runs a command in the background and notifies an Agor session upon completion.

**Usage:**
```bash
./agor-bg <session-id> <command> [args...]
```

### `agor-prompt`
Sends a prompt to a specific Agor session.

**Usage:**
```bash
./agor-prompt <session-id> <prompt>
```
